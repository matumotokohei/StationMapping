# StationMapping
各駅ごとの賃貸相場をgoogle mapのマーカーにて表示させるアプリです。
寒色から暖色へ0.5万刻みで色を変えています。
マーカーをタップすると駅名と家賃相場を表示します。
現在,playstoreへのリリース作業中です。
